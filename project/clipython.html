Python CLI — Hybrid Cryptography

Command-line tool for hybrid file and message encryption, with RSA/AES key management.

Python RSA AES CLI WIP

Overview

A Python command-line interface for hybrid cryptography — combining RSA asymmetric encryption for secure key exchange with AES symmetric encryption for fast data processing. Supports file and message encryption with an intuitive terminal UI.

🔐 Hybrid Encryption

RSA-2048 for key exchange, AES-256-CBC for payload. Best of both worlds: security and performance.

🗂️ File & Message

Encrypt/decrypt both text messages and binary files with a unified interface.

🔑 Key Management

Generate, export, and import RSA key pairs. Organized keystore for multi-recipient workflows.

💻 Terminal UI

Rich interactive CLI with menus, progress bars, and color-coded output for clear feedback.
